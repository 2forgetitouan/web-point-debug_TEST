# Outil de debug souris / clavier

## Lancer rapidement
```bash
python3 -m http.server 8000
```
Puis ouvrir `http://localhost:8000/index.html`.

## Fonctionnalités
- Affiche un point rouge à chaque clic souris.
- Log en console les coordonnées et le type de clic.
- Log en console les touches clavier pressées.
